# Customer Journey Analytics 호스팅 위치

Adobe Customer Journey Analytics은 북미, 유럽 및 APAC의 퍼블릭 클라우드 서비스 공급업체에서 제공하는 엔터프라이즈급 데이터 센터에 호스팅됩니다.

프로비저닝 시 고객은 Adobe Experience Platform 데이터가 위치할 지역을 지정합니다. Adobe Experience Platform 데이터 레이크에서 Customer Journey Analytics으로 수집된 데이터는 동일한 영역에 저장됩니다.

자세한 내용은 Adobe Experience Cloud 설명서의 [지역 데이터 수집](https://experienceleague.adobe.com/ko/docs/core-services/interface/data-collection/rdc)을 참조하십시오.


## 데이터가 저장된 데이터 센터 보기

>[!NOTE]
>
>한 데이터 센터에서 다른 데이터 센터로 데이터를 이동할 수 없습니다.

데이터가 저장된 데이터 센터를 보려면 다음 작업을 수행하십시오.

1. [Adobe Experience Cloud](https://experience.adobe.com)에 로그인합니다.

1. 인터페이스 오른쪽 상단의 앱 전환기 **[!UICONTROL 앱]**&#x200B;에서 ![Experience Platform](/help/assets/icons/Apps.svg)을(를) 선택합니다.

1. 지정된 데이터 센터에 대한 지역 코드가 Experience Platform의 오른쪽 상단에 표시됩니다.

   ![Experience Platform 홈 페이지의 데이터 센터](assets/experience-platform-data-center.png)

1. 지역 코드가 연관된 지리적 영역을 이해하려면 다음 표를 사용하십시오.

   | Adobe 지역 코드 | 클라우드 공급자 | 지리적 지역 |
   |-------------------|-------|-------------------------------------------|
   | VA7 | Azure | 미국(기본값) |
   | VA6 | AWS | 미국 (요청 시) |
   | NLD2 | Azure | 네덜란드, 암스테르담 |
   | CAN2 | Azure | 캐나다 중부, 토론토 |
   | AUS5 | Azure | 오스트레일리아, 시드니 |
   | GBR9 | Azure | 영국, 런던 |
   | IND2 | Azure | 인도 |
   | CHE2 | Azure | 스위스 |
