# 현재 Customer Journey Analytics(CJA) 릴리스 노트(2022년 4월)

>[!NOTE]
>
>이 페이지에는 변경될 수 있는 출시 전 정보가 포함되어 있습니다.

**마지막 업데이트**: 2022년 4월 19일

## 주요 기능

| 기능 | 설명 | [목표 날짜](/help/release-notes/releases.md) |
| ----------- | ---------- | ----- |
| Dimension 하위 문자열 | 차원 항목으로 사용할 문자열의 원하는 부분을 추출하는 여러 방법을 제공합니다. 이 기능을 사용하면 문자열에 구분된 값이 포함된 경우 문자열 차원을 배열로 처리할 수도 있습니다. [자세히 알아보기](../data-views/component-settings/substring.md) | 2022년 4월 20일 |
| Analytics 소스 커넥터에 대한 데이터 준비 | 다음 [Analytics 소스 커넥터](https://experienceleague.adobe.com/docs/experience-platform/sources/ui-tutorials/create/adobe-applications/analytics.html) 는 이제 [데이터 준비](https://experienceleague.adobe.com/docs/experience-platform/data-prep/home.html) Adobe Experience Platform에서 제공하는 기능. Adobe Real-time Customer Data Platform(RTCDP), CJA 및 Adobe Journey Optimizer(AJO) 고객은 이제 추가 필드 그룹으로 Analytics 필드 그룹을 확장할 수 있습니다. 또한 100개 이상의 데이터 준비 연산자를 활용하여 AEP(Adobe Experience Platform)으로 수집 중에 Analytics 데이터를 보강할 수 있습니다. 이제 RTCDP 고객이 프로필에 대해 여러 데이터 준비 사용 보고서 세트를 활성화할 수 있습니다.<p>Analytics 소스 커넥터를 통해 여러 보고서 세트를 수집하는 CJA 고객은 이제 보고서 세트 간 열 차이점을 제거할 수 있습니다. 예를 들어 &quot;검색어&quot;가 `eVar1` 한 보고서 세트 및 `eVar2` 다른 보고서 세트에서 데이터 준비를 사용하여 Analytics 필드 그룹을 두 eVar의 값을 병합하는 새 열로 확장할 수 있습니다. | 2022년 4월 27일 |
